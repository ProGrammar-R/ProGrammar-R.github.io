Help

This (de-)randomizer targets the original USA release of Azure Dreams for the Game Boy Color. You will need an image of your game pak in GBC format.

To use this site, first put your GBC file into the top box. Next, choose whatever options you want to use. If you cannot change any options, the link you clicked contained a seed (in which case you should use the same settings as the other person). Now click Randomize and a new GBC file will be created.

Do not overwrite your original GBC file; keep it somewhere safe or you may need to obtain it again from your original game pak.

About

The (de-)randomizer sets the each floor layout based off of the floor number and a random seed. As such, each trip up the tower should be unique but consistent with anyone else using the same seed.

This (de-)randomizer is a fan project and is not affiliated with Konami/KCE Nagoya, the developer and publisher of Azure Dreams (GBC). Please support the official release.
