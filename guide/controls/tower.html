General

L1/R1
Disable VSync if enabled (VSync is enabled by default). This is only possible once you have beaten the game and seen the message that appears when leaving Koh's house.
L2/R2
Re-enable VSync if disabled. This is only possible once you have beaten the game and seen the message that appears when leaving Koh's house.
